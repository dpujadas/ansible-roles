fcgiwrap
======

This role installs fcgiwrap, a CGI wrapper (commonly used for Nginx).

Package documentation: https://www.nginx.com/resources/wiki/start/topics/examples/fcgiwrap/

Role Variables
--------------

Example Playbook
----------------

    - hosts: servers
      roles:
        - { 
          role: fcgiwrap
        }
